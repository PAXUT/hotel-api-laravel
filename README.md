## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

# Hotel Booking System

## Laravel Scheduler Setup

### Windows (Using Task Scheduler)

1. Mở Task Scheduler(Window + R, nhập "taskschd.msc")
2. Tạo một task mới
Trong Task Scheduler, nhấp chuột phải vào "Task Scheduler Library" ở panel bên trái
Chọn "Create Basic Task"
Đặt tên cho task (ví dụ: "Laravel Scheduler")
Chọn tần suất chạy (chọn "Daily")
3. Trong tab "Actions", thêm action mới với:
   - Program/script: `php`
   - Add arguments: `E:\laragon\www\Hotel\api\artisan schedule:run`
   - Start in: `E:\laragon\www\Hotel\api`
4. Trong tab "Triggers", set trigger để chạy mỗi phút
5. Trong tab "Settings", chọn "Run whether user is logged in or not"

### Linux/Unix (Using Crontab)

Thêm dòng sau vào crontab:

```bash
* * * * * cd /path/to/your/project/api && php artisan schedule:run >> /dev/null 2>&1
```

## Kiểm tra Booking Hết Hạn

Hệ thống sẽ tự động kiểm tra và cập nhật trạng thái hết hạn cho các booking trong các trường hợp sau:

1. Đã qua ngày check-out và vẫn đang chờ duyệt (status_id = 3)
2. Chưa thanh toán và đã đến ngày check-in (chỉ áp dụng cho thanh toán COD)
3. Đã qua 24h kể từ ngày check-in nhưng chưa nhận phòng

Log của quá trình kiểm tra được lưu tại: `storage/logs/expired-bookings.log`
